# QuickSecure Mobile App

A React Native mobile application designed to enhance teacher safety in educational institutions. This app provides quick emergency response capabilities, real-time status updates, and seamless communication during critical situations.

## Features

- 🚨 Emergency Panic Button with 3-second hold activation
- 🔐 Biometric Authentication (Face ID/Touch ID)
- 📱 Phone Number Authentication Fallback
- 👤 User Profile Management with Photo Upload
- 🏫 Quick Actions for Common Scenarios:
  - Call Administrator
  - Fire Emergency
  - Medical Emergency
- 📋 Detailed Emergency Guidance
- 🔄 Real-time Status Updates
- 🔔 Push Notifications Support

## Technology Stack

- React Native with Expo
- TypeScript
- React Native Elements
- Expo Local Authentication
- Expo Image Picker
- AsyncStorage for Persistence
- React Native Gesture Handler

## Getting Started

### Prerequisites

- Node.js (v14 or newer)
- npm or yarn
- Expo CLI
- iOS Simulator or Android Emulator

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YourUsername/QuickSecure-Mobile-App.git
cd QuickSecure-Mobile-App
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npx expo start
```

4. Run on your preferred platform:
- Press `i` for iOS
- Press `a` for Android
